## User Flow Examples

### New User First Visit
1. Land on **Home** view
2. See overview and statistics
3. Click **Search** for quick lookup
4. Browse results
5. Explore **Medical Terms** for systematic learning
6. Check out **SOPs** for procedures

### Daily Reference Use
1. Click **Search** from bottom nav
2. Type medical term
3. Click result to view definition
4. Return to work with answer

### Training Session
1. Navigate to **SOPs**
2. Select procedure to learn
3. Work through steps one-by-one
4. Check off completed steps
5. Progress saves automatically
6. Resume later from same point

### Study Session
1. Go to **Medical Terms**
2. Expand category of interest
3. Read through terms
4. Open detail modals for deep learning
5. Use search to test recall
